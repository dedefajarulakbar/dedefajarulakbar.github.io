Fix High Ping & Packet Loss for Gaming: 7 Proven Steps

Last Updated: May 14, 2026 5 Min Read Gaming Network

High ping is not always your ISP's fault. Most lag comes from bad Wi-Fi, background downloads, and wrong DNS settings. I dropped my Valorant ping from 180ms to 38ms in Aceh using these exact steps — no VPN needed.

1. Use Ethernet, Not Wi-Fi

Wi-Fi adds 15-40ms of jitter. A $10 CAT6 cable gives you stable ping instantly. If you must use Wi-Fi, use 5GHz band and sit within 3 meters of the router.

2. Change DNS to Cloudflare or Google

Your ISP DNS is slow. Switch to faster DNS to reduce initial connection time.

  1. Windows: Settings → Network → Change adapter options → Right-click Wi-Fi/Ethernet → Properties
  2. Select IPv4 → Properties → Use: 1.1.1.1 and 1.0.0.1 (Cloudflare)
  3. Or use Google: 8.8.8.8 and 8.8.4.4

3. Close Background Bandwidth Hogs

Press Ctrl+Shift+Esc → Performance → Open Resource Monitor → Network. Kill:

  • Windows Update
  • Steam/Epic downloads
  • OneDrive, Google Drive sync
  • Chrome/YouTube streaming on other devices
My Rule: Before ranked game, I run netsh interface ip show config in CMD to check if anything is downloading. Saves me from 200ms spikes.

4. Set Your Game to High Priority

  1. Launch game
  2. Alt+Tab → Task Manager → Details
  3. Right-click game.exe → Set priority → High
  4. Do same for anti-cheat (vgc.exe, etc)

5. Use Ping Test to Find Best Server

Don't auto-select. Test manually in CMD:

ping sg.valorant.com -t (Singapore)
ping 103.28.54.1 -t (Mumbai)

Pick server with lowest stable ms, not lowest average. Stable 50ms beats jumping 30-90ms.

6. Restart Router the Right Way

Unplug router 30 seconds, not 5. This clears ISP cache and gets new IP route. Do this once a week. I do every Sunday night.

7. Limit Other Devices (QoS)

Login to router (192.168.1.1) → Find QoS or Bandwidth Control → Set your PC/phone MAC address to High priority. This stops family Netflix from killing your ping.

Avoid VPNs for Gaming: VPNs add 20-50ms unless your ISP routing is broken. Test ping with and without VPN. In Indonesia, VPN usually makes it worse.

What Ping Should You Expect?

  • Java/Sumatra to Singapore: 30-50ms (good)
  • To Japan: 80-110ms (playable)
  • Over 150ms: fix your network first

Stop Windows 11 24H2 Auto Update: 3 Working Methods

Last Updated: May 14, 2026 6 Min Read Windows 11

Windows 11 24H2 has been forcing itself onto PCs since late 2024, breaking drivers, games, and work apps. Microsoft keeps pushing it even if you pause updates.

Here are three methods that actually work in 2026 to block 24H2 while still getting security patches. No registry hacks that break your PC.

Important: Do NOT disable Windows Update completely. You will miss critical security patches. These methods block feature updates (like 24H2) but allow security updates.

Method 1: TargetReleaseVersion (Best for Pro & Home)

This tells Windows to stay on your current version like 23H2.

  1. Press Win + R, type regedit, press Enter.
  2. Navigate to: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate
  3. Right-click → New → DWORD (32-bit) Value. Name it TargetReleaseVersion. Set value to 1.
  4. Right-click → New → String Value. Name it TargetReleaseVersionInfo. Set value to 23H2.
  5. Restart PC. Windows will stay on 23H2.

Method 2: Group Policy (Windows 11 Pro Only)

  1. Press Win + R, type gpedit.msc.
  2. Go to Computer Configuration → Administrative Templates → Windows Components → Windows Update → Manage updates offered from Windows Update.
  3. Double-click "Select the target Feature Update version".
  4. Select Enabled. Type 23H2 in the box.
  5. Apply → OK → Restart.

Method 3: Metered Connection Trick (Easiest)

Windows won't download big updates on metered connections.

  1. Settings → Network & Internet → Wi-Fi (or Ethernet).
  2. Click your network → turn on Metered connection.
  3. Go to Windows Update → Advanced options → turn off "Download updates over metered connections".
My Recommendation: Use Method 1. It has worked for me for 18 months. I stayed on 23H2 while getting security updates every month. When 25H2 is stable in late 2026, I'll change the registry to 25H2.

How to Check If It Worked

Go to Settings → Windows Update → Update history. You should see security updates but NO "Feature update to Windows 11 24H2". If you see it, the block failed.

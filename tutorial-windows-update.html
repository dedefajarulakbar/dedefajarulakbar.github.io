How to Stop Windows Update from Breaking Your PC

Last Updated: May 14, 2026 4 Min Read Windows 10 & 11

Automatic Windows Updates can restart your PC during important work, install buggy drivers, and break software you rely on. For a stable computer, the best approach is to set updates to manual and check them once per week.

You stay secure without the surprise restarts.

Important: Never disable Windows Update completely. Security patches are critical to protect your PC from viruses and hackers. This guide shows you how to control WHEN updates install, not IF they install.

Method 1: Pause Updates - Best for Most People

Windows lets you pause all updates for up to 5 weeks. This is the safest and easiest method.

  1. Press Win + I to open Settings.
  2. Click Windows Update in the left sidebar.
  3. Next to "Pause updates", click the dropdown menu.
  4. Select Pause for 5 weeks.
  5. Set a calendar reminder to manually check for updates every Friday.
My Workflow: I pause updates on Monday morning. Every Friday at 5 PM, I click "Resume updates" and let everything install overnight. I’ve had zero work interruptions in 2 years using this method.

Method 2: Set Active Hours + Metered Connection

This prevents restarts during your work hours and stops Windows from auto-downloading large updates.

  1. In Windows Update, click Advanced options.
  2. Under "Active hours", select Manually.
  3. Set your work hours. For example: 8:00 AM to 6:00 PM. Windows will not restart during this time.
  4. Go to Settings → Network & internet → Wi-Fi.
  5. Click your Wi-Fi network name, then turn on Metered connection.
  6. Windows will not download large feature updates on a metered connection unless you click "Download" manually.

How to Manually Check for Updates Safely

Do this once per week, preferably Friday evening or weekend.

  1. Press Win + I → Windows Update.
  2. Click Check for updates.
  3. Install all Security Updates and Cumulative Updates immediately.
  4. You can delay Feature Updates like "Windows 11, version 24H2" for 2-3 weeks to avoid bugs.
  5. Click Restart now only when you are done working for the day.

Frequently Asked Questions

Will I miss security updates?

No, if you check manually once per week. Microsoft releases security updates on "Patch Tuesday", the second Tuesday of each month. Checking every Friday means you're only 3 days behind, which is safe.

Why not disable the Windows Update service?

Disabling the service in services.msc breaks the Microsoft Store, Xbox app, and other core apps. The "Pause" method is official and won't break anything.

An update broke my PC. How do I fix it?

Go to Settings → Windows Update → Update history → Uninstall updates. Find the most recent update, usually "KB" followed by numbers, and click Uninstall. Then pause updates for 2 weeks.

Should I install "Optional updates" and "Driver updates"?

Only if something is broken. If your Wi-Fi, sound, or printer works fine, skip driver updates. They cause 80% of update problems. Only install them if a device stops working.
